經歷(current) 技能(current) 參與專案(current)

網頁爬蟲

使用 Python 函式庫撰寫網頁爬蟲程式，並將資料匯入資料庫；在 Linux 作業系統上設定 Crontab，對爬蟲程式進行工作排程，滿足定時且自動爬取網頁內容的需求。

資料清洗及探勘

使用 Python 之 Pandas、Numpy 函式庫進行資料預處理及統計運算，並利用 Scikit-learn 函式庫的線性迴歸及邏輯迴歸模型進行機器學習。

資料視覺化

使用 PHP 串接後端資料庫，並以 jQuery 函式庫將資料傳入至前端，透過 Apache ECharts、Chart.js 等 Javascript 庫搭配 Bootstrap 前端框架滿足資料視覺化的需求。

Web API

使用 Flask 框架撰寫 Web API 並達成對資料庫進行 CURD 操作的需求。

關聯式資料庫

具備在 CentOS 作業系統上架設 MariaDB 以及 phpMyAdmin 資料庫管理工具的能力，並熟悉 SQL 指令以進行資料庫運維以及資料處理。

非關聯式資料庫

具備在 CentOS 作業系統上架設 MongoDB 及 InfluxDB 的能力，並熟悉資料庫運維以及資料探勘之相關指令。

ELK

具備建置 Elasticsearch、Logstash 及 Kibana 的能力，並使用其達成資料蒐集、預處理、探勘及視覺化的需求。

Hadoop

具備建置 Hadoop 叢集系統的能力，並使用 Hive 進行資料預處理及探勘。

商業智慧軟體

熟悉 Tableau 及 Splunk 兩項商業智慧軟體在資料處理上的相關操作。
